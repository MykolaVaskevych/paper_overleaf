# FYP Thesis - A Comparative Analysis of DQN and Actor-Critic RL Algorithms

This repository contains the LaTeX source files for my Final Year Project (FYP) thesis at University of Limerick.

> **Note:** This repository is a submodule of my [Final Year Project (FYP)](https://github.com/MykolaVaskevych/fyp).

## Project Overview

This thesis compares and contrasts two families of Reinforcement Learning algorithms:
- **Value-Based Methods**: Deep Q-Network (DQN)
- **Policy Gradient Methods**: Actor-Critic (A2C, PPO)

The research evaluates these algorithms on Atari and classic control environments, measuring convergence speed, sample efficiency, and training stability.
